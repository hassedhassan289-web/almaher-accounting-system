# نظام الماهر المحاسبي 📊

## نظام محاسبي شامل ذكي لإدارة المبيعات والمشتريات والمخزون والصناديق

### ✨ الميزات الرئيسية:
- ✅ إدارة مبيعات متقدمة (كاشير سريع + فواتير يدوية)
- ✅ إدارة المشتريات والموردين
- ✅ تتبع المخزون والأصناف
- ✅ نظام صناديق مترابط (كاشير + خزنة + مديونية)
- ✅ إدارة ديون العملاء والسداد الآجل
- ✅ تقارير شاملة وصافي أرباح
- ✅ نسخ احتياطي تلقائي واستعادة
- ✅ دعم عربي كامل

### 🛠️ المتطلبات:
- Python 3.11+
- PyQt6
- SQLite (مدمج)

### 🚀 البدء:
```bash
pip install PyQt6
python main.py
```

### 📁 هيكل المشروع:
