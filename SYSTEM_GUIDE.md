## 🚀 كيفية البدء:

### 1️⃣ **تثبيت البرنامج**:
```bash
# استنساخ المشروع
git clone https://github.com/hassedhassan289-web/almaher-accounting-system.git
cd almaher-accounting-system

# تثبيت المكتبات
pip install -r requirements.txt

# تشغيل البرنامج
python main.py
```

### 2️⃣ **البيانات الأولية**:
- العملة: **الريال اليمني (YER)**
- الصناديق الافتراضية:
  - 💵 صندوق النقدية الكاشير
  - 🏦 الخزنة الرئيسية
  - 📊 صندوق المديونية

### 3️⃣ **الأقسام الرئيسية**:

#### 🏪 **المبيعات**:
- **كاشير سريع**: مسح باركود وإضافة سريعة
- **فاتورة يدوية**: إدخال يدوي متقدم مع تعديل الأسعار
- **المرتجعات**: نظام صارم مع التحقق من IMEI
- **السجل**: تاريخ كامل بالفواتير والإحصائيات

#### 💰 **الصناديق والحسابات**:
- **الصناديق**: عرض الأرصدة الحالية لجميع الصناديق
- **التحويلات**: تحويل أموال آمن بين الصناديق
- **السحب والإيداع**: عمليات مالية مفصلة
- **جرد الوردية**: تقرير شامل نهاية اليوم مع الفائض/العجز

#### 💳 **الديون والمديونية**:
- **العملاء المديونين**: قائمة كاملة مع الديون المتأخرة
- **سند السداد**: إصدار سندات رسمية مع طباعة
- **الفواتير الآجلة**: تتبع كامل للفواتير المعلقة
- **تقارير الديون**: تحليل الديون المتأخرة والمستحقة

#### 📊 **التقارير الشاملة**:
- **الأرباح والخسائر**: شامل ودقيق بالفترة الزمنية
- **جرد الصناديق**: تقرير تفصيلي لكل صندوق
- **تتبع IMEI**: سجل كامل لكل جهاز هاتفي
- **ميزان المراجعة**: فحص دقة الحسابات
- **المخزون**: قيمة المخزون الإجمالية والأصناف الناقصة

## 🔧 تخصيص النظام:

### تغيير العملة:
```python
# في config.py
CURRENCY_NAME = "الريال اليمني"
CURRENCY_CODE = "YER"
CURRENCY_SYMBOL = "ر.ي"
```

### تغيير أسعار الصرف:
```python
# في config.py
EXCHANGE_RATES = {
    "YER": 1.0,
    "USD": 250.0,  # تحديث السعر هنا
    "SAR": 66.0,
}
```

### تغيير مسار النسخ الاحتياطية:
```python
# في config.py
BACKUP_DIR = "C:/MyBackups"  # أو أي مسار آخر
```
